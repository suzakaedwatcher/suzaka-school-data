# 須坂モデルはどこから来たのか

調査日: 2026-03-23

## 要約

須坂市教育委員会が2022年に作成した「須坂モデル」（小中一貫・義務教育学校化の構想）は、市民の提言から生まれたものではありません。
2021年2月の**国立教育政策研究所（NIER）講演会**の内容と酷似しており、国の推進パッケージをローカライズしたものです。

一方、文科省が2015年に策定した「手引き」の**第4章（小規模校を存続させる場合の教育の充実）**は検討された形跡がありません。

---

## 1. 時系列で見る「小中一貫」の出現

| 時期 | 出来事 | 「小中一貫」の記載 |
|------|--------|-------------------|
| 2015年1月 | 文科省「手引き」策定 | 統合も存続も**両方**の選択肢を提示 |
| 2020年3月 | 須坂市「学校施設長寿命化計画」 | 15校を80年使う方針 |
| 2020年度 | あり方検討会議の提言 | **なし**（「育みたい資質能力」） |
| **2021年2月** | **NIER講演会「9年間の学び舎を作ろう」** | **小中一貫教育校の推進一色** |
| 2021年度 | 子どもの学びのあり方検討会議の提言 | **なし**（「非認知能力」「個別最適な学び」） |
| **2022年度** | **教育委員会が「須坂モデル」を作成** | **ここで初めて小中一貫が登場** |
| 2022〜23年度 | 審議会で須坂モデルを前提に議論 | 義務教育学校化を答申 |
| 2024年 | 学園構想 基本方針案 | 15校→4学園 |

**注目すべきポイント:** 2つの検討会議（2020〜2021年度）の提言には「小中一貫」の文言が一切ありません。教育委員会は「検討会議の提言を具体化した」と説明していますが、検討会議が言っていない「小中一貫」がなぜ出てきたのでしょうか。

---

## 2. NIER講演会と須坂モデルの対照

### 講演会の概要

- **正式名称**: 令和2年度 国立教育政策研究所 文教施設研究講演会
- **タイトル**: 「9年間の学び舎を作ろう —学校、地域、行政、設計者 対話で生まれる学校建築—」
- **日時**: 2021年2月10日（オンライン）
- **主催**: 国立教育政策研究所（NIER）＝文部科学省の研究機関

### 内容の対照表

| NIER講演会（2021年2月） | 須坂モデル（2022年） |
|---|---|
| 9年間を見通した教育課程 | 9年間を見通したカリキュラム |
| 中1ギャップの解消 | 中1ギャップの解消 |
| 非認知能力の育成 | 非認知能力の育成 |
| 義務教育学校の設置 | 義務教育学校の設置 |
| 施設一体型/分離型 | 施設一体型/分離型 |
| 4-3-2制のステージ区分 | 4-3-2制のステージ区分 |
| アクティブ・ラーニング | アクティブ・ラーニング |

講演資料は小中一貫教育校の推進一色です。成果データも肯定的なもののみ（「中学校への進学に不安を覚える児童が減少した 96%」等）。**小規模校存続の選択肢には一切触れていません。**

須坂市教育委員会はこの講演会資料を入手・保存しています（112ページ、27.7MB）。

## 3. 文科省「手引き」の無視された第4章

須坂市は講演会の推進部分を採用する一方で、手引きの以下の記述を検討した形跡がありません。

### 手引き p.5（本手引の位置付け）

> 「本手引の内容を**機械的に適用することは適当ではなく**、飽くまでも各市町村における主体的な検討の**参考資料**として利用されることが望まれます」

### 手引き p.3（地域コミュニティの核としての性格への配慮）

> 「**行政が一方的に進める性格のものでないことは言うまでもありません**」

### 手引き 第4章（p.33-38）「小規模校を存続させる場合の教育の充実」

統合を選択しない主なケースとして、手引きは以下を挙げています:

1. 離島や山間部、豪雪地帯など、安全安心な通学ができないと判断される場合
2. 更なる少子化でさらなる統合が進められない場合
3. 同一市町村内で統合不可能な場合
4. **学校を当該地域コミュニティの存続や発展の中核的な施設と位置付け、地域を挙げてその充実を図ることを希望する場合**

さらに小規模校のメリットとして:

- 一人一人の学習状況を的確に把握でき、補充指導や個別指導が行いやすい
- 意見や感想を発表できる機会が多くなる
- 一人一人がリーダーを務める機会が多くなる
- 運動場や体育館、特別教室などが余裕をもって使える
- 地域の協力が得られやすく、郷土の教育資源を最大限に生かした教育活動が展開しやすい

**須坂の検討プロセスにおいて、この第4章が検討された議事録や資料は見つかっていません。**

## 4. 国のダブルスタンダード

同じ文科省/NIERが、異なるメッセージを発信しています:

| 発信元 | メッセージ |
|---|---|
| **手引き**（2015年） | 「統合も存続も両方あり。機械的に適用するな」 |
| **NIER講演会**（2021年） | 「9年間の学び舎を作ろう！」統合推進一色 |
| **手引き改訂素案**（2026年） | 「結論ありきで評価することを避けなければならない」 |

須坂市は手引きの第4章（存続）を読まず、講演会の推進部分だけを採用しました。ただし、国の発信自体がダブルスタンダードだったという構造的問題もあります。

---

## 5. この分析の意味

| 論点 | 評価 |
|------|------|
| 教育委員会に悪意はあったか | おそらくない。国の推進策に忠実に従った行動 |
| 問題は何か | **手引きの4章を無視し、講演会の推進部分だけを採用した選択的な情報収集** |
| 市民の声の扱い | 検討会議の提言（小中一貫なし）を「具体化」する過程で、提言にない内容を注入 |
| 最も効果的な問いかけ | **「手引きの第4章を検討しましたか？」** |

教育委員会への問いは「統合は正しいのか」ではなく、**「なぜ手引きが示した存続の選択肢を検討しなかったのか」**です。
