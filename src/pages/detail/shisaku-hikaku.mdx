# 長野県4市の教育施策比較 — 開発手法で読み解く

調査日: 2026-03-22

教育改革の進め方を、ソフトウェア開発の手法になぞらえて比較します。
「何をやるか」だけでなく **「どう進めるか」** に注目すると、各市の教育行政の性格がはっきり見えてきます。

---

## 開発手法の早わかり

| 手法 | 考え方 | 強み | リスク |
|------|--------|------|--------|
| **リーン** | 小さく試して、検証してから広げる | 失敗コストが小さい。方向修正が容易 | 大きなビジョンが見えにくい |
| **アジャイル** | 課題ごとにチームを組み、反復的に改善する | 現場の声が反映されやすい | 全体最適になりにくい |
| **ウォーターフォール** | 全体設計を先に決め、順番に実行する | 完成形が明確。大規模な変革に向く | 途中で止められない。失敗が大きい |
| **その場しのぎ** | 問題が起きてから対処する | 平時のコストが低い | 手遅れになるリスク。学びが蓄積しない |

---

## 松本市 — リーン（小さく試して、広げる）

> **「統合ありき」ではなく「子どもの選択肢を増やす」**

### 施策の全体像

| 施策 | 開始 | 内容 |
|------|------|------|
| 特認校制度 | 2022年〜 | 安曇小中で開始。成果を見て2024年に中山小へ拡大 |
| デュアルスクール | 2023年〜 | 県外から住民票移動なしで山間部校に短期就学（奈川小中・大野川小中）。令和6年度は9世帯利用 |
| インクルーシブセンター | 2024年〜 | 乳幼児〜高校卒業まで切れ目ない発達支援 |
| フリースクール補助金 | 2025年〜 | 月額上限15,000円。不登校の子どもの選択肢を広げる |
| フリースペース | 2024年〜 | 市内7地区公民館に若者の居場所を開設 |
| 全校特色化「みんなの未来の学校」 | 2025年〜 | 教育長所信表明で方針を掲げ、全小中学校の特色化を推進 |
| 市立特別支援学校 | 準備中 | 新校舎2029年度完成予定 |

### なぜ「リーン」か

1. **小さく始めて検証する**: 特認校は安曇小中1校で開始→成果を確認してから中山小に拡大。デュアルスクールも2校で試行中
2. **失敗コストが小さい**: 制度の変更・廃止で修正可能。建物を建てる必要がない
3. **選択肢の並行提供**: 特認校・デュアルスクール・フリースクール補助・フリースペースなど、子どもの状況に応じた複数の道を用意
4. **ソフトウェア重視**: ハコモノ（施設統合）ではなく、制度設計・教員支援・授業改革に投資
5. **共有ビジョンの下で分散実行**: 「学都松本」という方向性を示し、各校が自律的に特色化を進める

### 小規模校への対応

統廃合ではなく**魅力化**が基本路線。山間部の小規模校を「少人数のきめ細やかな学び」「自然環境」として積極的にブランディング。不登校気味の児童が都市部の大規模校から小規模校へ移るケースも多い。

### 教育長の姿勢

曽根原好彦教育長（2025年4月〜）の所信表明:
- インクルーシブ教育の継承・発展
- 全校の特色化「「みんなの未来の学校」」
- 教員の働き方改革（5時間授業の日を増やし余裕を創出）
- 「学校教育の当たり前の見直し」（校則、宿題、評価方法）を各校に促す

## 佐久市 — アジャイル（地区ごとにスプリント）

> **課題が出てから、地区ごとに住民と一緒に考える**

### 実績と進行中の施策

| 地区 | 課題 | 対応 | 期間 |
|------|------|------|------|
| 臼田地区 | 児童減（小規模校4校） | 4校→1校に統合。新校舎建設 | 2014年〜2023年（9年） |
| 浅間地区 | 児童増（佐久平駅周辺の人口増） | 通学区見直しの検討中 | 2024年〜（継続中） |

### なぜ「アジャイル」か

1. **課題ごとにチームを組む**: 臼田と浅間はまったく別のプロセス（児童減 vs 児童増）。同じ市内でも正反対の課題に、別々の検討委員会で対応
2. **住民参加のイテレーション**: 検討委員会（25名規模）→パブリックコメント→住民説明会→グループワーク。臼田は16回の会議を重ねて合意形成
3. **全体計画を持たない**: 市全体の「学校再編マスタープラン」は存在しない。課題が顕在化した地区から個別に着手
4. **長い検討期間**: 臼田は構想から開校まで9年。拙速を避ける文化
5. **施設は長寿命化が基本**: 既存校舎を長持ちさせる方針で、統廃合は最終手段

### 保守的な面

先進的な教育施策（特認校・コミュニティスクール・小中一貫など）は**未導入**。「教育の質的改革」よりも「量的調整（統合・通学区変更）」に対応が限定されている。教育振興基本計画「コスモスプラン」は理念的だが、松本市のような具体的な新制度の試行には至っていない。

### 財政面

教育施設投資は1人あたり17,518円で**県内3位**（須坂市の3.4倍）。教育費に占める施設投資の割合は28.4%で、必要な施設整備にお金を使えている。

## 長野市 — その場しのぎ（問題が起きてから対処）

> **児童が限界まで減るのを待ち、地域が「もう無理」と言ってから動く**

### 近年の閉校ラッシュ

| 学校 | 閉校年 | 経緯 |
|------|--------|------|
| 信更中学校 | 2023年3月 | 最後の卒業生6名で閉校。74年の歴史 |
| 信更小学校 | 2024年3月 | 2016年に2校を統合→8年で再閉校。最終年度14名 |
| 七二会中学校 | 2024年3月 | 77年の歴史 |
| 清野小学校 | 2025年3月 | 130年超の歴史。保護者が「閉校やむなし」 |
| 鬼無里中学校 | 2027年3月予定 | 唯一の特認校対象が閉校へ |

### なぜ「その場しのぎ」か

1. **包括計画がない**: 学校適正規模・適正配置の全市計画を策定していない。75校（小52＋中23）を抱えながら、将来像を示していない
2. **待ちの姿勢**: 大岡小12人、中条小22人など教育環境として限界の学校があるが、市教委から動かない。地域の自治協議会が「閉校やむなし」と要望するのを待つ
3. **学びが蓄積しない**: 信更小は統合わずか8年で再閉校。最初の統合判断が甘かったが、その教訓が他地区に活かされている形跡がない
4. **試行がない**: 特認校は鬼無里1校のみ。その鬼無里中も閉校予定で、制度として発展する見込みがない
5. **隣接学校選択制度は対症療法**: 大規模校5校から隣接校への選択を認める制度があるが、小規模校の延命策にはなっていない

### 規模の問題

長野市は人口37万人、小学校52校。須坂市とは規模が大きく異なる。しかし**規模が大きいからこそ包括計画が必要**とも言える。中山間地域に100名以下の小学校が8校ある現状に対し、戦略的な対応を示していない。

## 須坂市 — ウォーターフォール（全体設計を先に、順番に実行）

> **15校→4学園の設計図を先に描き、住民説明は「後工程」**

### 学園構想の概要

| 項目 | 内容 |
|------|------|
| 計画 | 15校（小11＋中4）→ 4学園（義務教育学校） |
| 施設形態 | 一体型＋分離型の混在 |
| タイムライン | 2026年〜段階的に開校（具体的スケジュールは未公表） |
| 意思決定 | 教育委員会が構想を策定→住民に説明 |
| 費用 | 未公表 |

### なぜ「ウォーターフォール」か

1. **要件定義が先**: 「4学園・義務教育学校」という結論が先にあり、住民の要望からボトムアップで出てきたものではない
2. **全体を一括で設計**: 4中学校区すべてを同時に再編する計画。地区ごとの段階的実施ではない
3. **工程が順番**: 構想策定→住民説明→設計→建設→開校。前の工程に戻りにくい
4. **大規模投資が前提**: 施設建設・改修は不可逆。建ててしまったら「やっぱりやめる」ができない
5. **テストなし**: 1校で試行して成果を検証してから拡大、というプロセスがない

### ウォーターフォールのリスク

- **要件の誤り**: 「人口5万人規模で4学園を同時に義務教育学校化した成功事例は全国に存在しない」（[成功事例調査](/detail/gimu-seikou)より）
- **手戻りコスト**: 施設建設後に「分離型では一貫教育が形骸化する」と判明しても修正不能
- **ステークホルダーの離脱**: 住民アンケートで統合を望む声は1.9%。合意なきまま進めれば頓挫のリスク
- **教員減少**: 統合により教職員定数が6〜8人減る構造的問題（[財政データ](/solutions/5)参照）

## この比較から見えること

### 松本市が示す「もう一つの道」

松本市は須坂市と同じ長野県内にありながら、正反対のアプローチを取っている。

- 須坂市: ハコモノ（施設統合）で問題を解決しようとする
- 松本市: ソフト（制度・人・プログラム）で選択肢を増やす

松本市の特認校制度もデュアルスクールも、**既存の校舎をそのまま使っている**。新しい建物は建てていない。それでも小規模校に児童を呼び込み、教育の選択肢を広げている。

### 佐久市が示す「合意形成のコスト」

佐久市の臼田地区統合は**9年**かかった。16回の検討委員会、住民説明会、パブリックコメントを経て、ようやく合意に至った。これは「時間がかかりすぎる」のではなく、**合意形成とはそういうものだ**ということを示している。

須坂市が4学園を同時に進めようとすれば、この何倍もの合意形成コストがかかる。

### 長野市が示す「何もしないリスク」

長野市のように「問題が限界に達するまで待つ」アプローチにも大きなリスクがある。大岡小の児童12人は、十分な教育環境にあるとは言いがたい。何もしないことは、**現在の子どもたちの教育機会を犠牲にしている**可能性がある。

須坂市が「何かしなければ」と考えること自体は正しい。問題は**「何を」「どう」やるか**だ。

### 須坂市への処方箋

ウォーターフォールからリーンへの転換:

1. **まず1校で試す**: 最も課題が切迫している地区（例: 豊丘小）で特認校制度を試行する
2. **検証してから広げる**: 1〜2年の成果を見てから、次の施策を判断する
3. **選択肢を複数用意する**: 義務教育学校だけでなく、小中一貫型小・中学校、特認校、合同授業（山県方式）など、地区の実情に合った方法を選ぶ
4. **費用を公開する**: 4パターンそれぞれの費用を公開し、市民が判断できるようにする
5. **不可逆な投資は最後に**: 施設建設は最終段階。まずソフト面の施策で成果を出す
