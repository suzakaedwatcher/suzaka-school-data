# 小中一貫教育は本当に効果があるのか

NIER（国立教育政策研究所）の全国調査を中心に、小中一貫教育の効果に関するエビデンスを整理します。

---

## 1. NIER調査の概要

| 項目 | 内容 |
|------|------|
| 正式名称 | 「小中一貫教育の成果と課題に関する調査研究」 |
| 実施機関 | 国立教育政策研究所（文科省の研究機関） |
| 調査時期 | 2014〜2015年度 |
| 対象 | 全国の小中一貫教育実施校 **1,130校** |
| 回答者 | 学校管理職（校長等） |
| 原文 | [NIER調査報告書（PDF）](https://www.nier.go.jp/05_kenkyu_seika/pdf_seika/h27/1-1_all.pdf) |

これが現時点で最も大規模な小中一貫教育の効果調査です。

---

## 2. 主な結果

### 肯定的な結果

- **約6割の学校**が「全国学力・学習状況調査の結果が向上した」と回答
- **96%の学校**が「中学校進学への不安が減少した」と回答
- 学習習慣の改善、生徒指導上の問題の減少なども報告

### 課題として報告された結果

- 業務軽減策を実施していた学校は**約2割**にとどまる
- 教職員の**打ち合わせ・会議の増加**が広く指摘
- 小学校と中学校の**教育文化の違い**による現場の困難

---

## 3. この調査をどう読むべきか

### エビデンスとしての限界

この調査には重要な限界があります。

**1. 自己申告ベース**

「学力が向上した」と回答しているのは**学校の管理職**です。導入を決定した側の主観的評価であり、第三者による客観的な測定ではありません。

**2. 比較対照群（統制群）がない**

小中一貫を導入しなかった学校との比較がありません。同時期に全国的に学力が向上していた場合、小中一貫の効果なのか時代の変化なのか区別できません。

**3. 因果関係の証明ではない**

「導入した → 学力が上がった」は相関であり、因果関係の証明ではありません。小中一貫に積極的に取り組む自治体はそもそも教育予算が多い、教員の意欲が高いなど、他の要因が重なっている可能性があります。

### 何が言えて何が言えないか

| 言えること | 言えないこと |
|-----------|-------------|
| 導入校の多くが「手応え」を感じている | 客観的に学力が向上したかどうか |
| 中学進学への不安軽減は広く報告されている | 不安軽減と学力向上が関連しているか |
| 教職員の負担増は広く発生している | 負担増が教育の質にどう影響するか |
| 効果は自治体によってばらつきがある | どの条件で効果が出るか |

## 4. 他の研究・データ

### 中央教育審議会での議論

[中教審での主な意見](https://www.mext.go.jp/b_menu/shingi/chukyo/chukyo3/002/siryo/attach/1353442.htm)では、制度化に際して以下の懸念が示されています。

- **人間関係の固定化**: 9年間同じメンバーで過ごすことで逃げ場がなくなる
- **転校時の問題**: 9年制の途中で転入・転出する児童生徒の不適応
- **6-3制の利点の喪失**: 小学校卒業・中学校入学という節目がもたらす成長機会
- **受験エリート校化**: 公立の義務教育学校が特定層に選ばれる学校になるリスク

### 保護者向けの解説

- [ベネッセ「義務教育学校とは？小中一貫校との違い」](https://benesse.jp/educational_terms/36.html) — メリット・デメリットの平易な解説
- [ソクラテスのたまご「小中一貫校のメリット・デメリット」](https://soctama.jp/column/73190) — 保護者視点での両面解説

## 5. 須坂市への示唆

須坂市教育委員会は小中一貫教育の「メリット」を説明していますが、以下の点が未対応です。

1. **須坂市のデータに基づく効果予測がない** — 全国調査の「約6割が成果を感じた」がそのまま須坂に当てはまる保証はない
2. **デメリットとその対策が示されていない** — NIER調査自体が課題を報告しているにもかかわらず
3. **統合しない場合の教育改善策と比較されていない** — 小中一貫以外の方法（ICT活用、合同授業、教科担任制の単独導入等）との比較検討がない

なお、須坂モデルが国の推進パッケージのコピーであることについては [NIER講演会と須坂モデルの関係](/detail/nier-analysis) で分析しています。
