# 義務教育学校・小中一貫教育 全国成功事例調査

調査日: 2026-03-16

## 1. 施設一体型 義務教育学校の成功事例

### 1-1. 信濃町立信濃小中学校（長野県信濃町）

| 項目 | 内容 |
|------|------|
| 自治体 | 長野県上水内郡信濃町（人口約8,000人） |
| 開校年 | 2012年4月（小中一貫校）→ 2016年4月（義務教育学校に移行） |
| 統合規模 | 小学校5校＋中学校1校 |
| 児童生徒数 | 約400名（開校時）→ 減少傾向 |
| 学年制 | 4・5制（1-4年＝初等部、5-9年＝高等部） |
| 施設形態 | 施設一体型 |

**成功している点:**
- 全国学力学習状況調査において学力的成果
- 9年間を見通したカリキュラムの実現
- 5年生からの教科担任制
- 時事通信社「教育奨励賞」努力賞受賞

**成功の条件:**
- 準備に7年余りの期間を確保
- 町が唯一の中学校区であり、統合の構造がシンプル
- 人口8,000人の小規模自治体

**課題:**
- 少子化による部活動の維持困難（部員3名程度の部も）
- 通学にスクールバス・デマンドタクシーが必要
- 2019年の検証報告書で「課題も多くあげられている」

**須坂との比較:**
| 項目 | 信濃町 | 須坂市 |
|------|--------|--------|
| 人口 | 約8,000人 | 約50,000人 |
| 統合前 | 6校 | 15校 |
| 統合後 | 1校 | 4学園 |
| 中学校区 | 1つ | 4つ |
| 施設形態 | 一体型 | 一体型＋分離型混在 |
| 準備期間 | 約7年 | 未定 |

→ 信濃町は「人口8,000人・中学校区1つ・選択肢なし」。須坂市への直接適用は限定的。

| 児童生徒数 | 約1,200名 |

## 2. 施設分離型の成功事例

### ★ 三鷹市のコミュニティ・スクール型小中一貫教育（最重要事例）

| 項目 | 内容 |
|------|------|
| 自治体 | 東京都三鷹市（人口約19万人） |
| 導入年 | 2006年「にしみたか学園」→ 2009年全7学園に拡大 |
| 学園数 | 7学園（市内全小中学校が参加） |
| 施設形態 | **全学園が施設分離型** |

**成功している点:**
- 施設分離型でありながら小中一貫教育の成果を実現
- 市独自の学力テストで学力向上傾向
- 中学生の不登校出現率の減少
- コミュニティ・スクールを基盤とした地域参画型運営
- 全国から視察が相次ぐモデルケース

**具体的な工夫:**
- 小中教員の相互乗り入れ授業
- 小中一貫カリキュラムに基づく9年間の系統的指導
- コミュニティ・スクール活動による地域住民の参画

**須坂への示唆:**
- **施設分離型でも成果は出せる最大の実証例**
- ただし「義務教育学校」ではなく「小中一貫型小・中学校」として実施
- **統廃合を伴わないため教職員数の減少がない**

## 3. 成果データ

### 文科省実態調査（平成26年、1,130校対象）

- 「大きな成果が認められる」: 10%
- 「成果が認められる」: 77%
- 合計87%が「成果あり」

**重要な留意点:**
- **自己評価**であり、対照群との比較研究ではない
- 国立教育政策研究所:「中1ギャップの前提となる事実認識は客観的事実とは言い切れない」

### 施設形態の全国構成比

| 施設形態 | 割合 |
|---------|------|
| 施設一体型 | 13% |
| 施設隣接型 | 5% |
| **施設分離型** | **78%** |

→ 小中一貫教育の大多数は施設分離型で実施されている。

### 文科省見解

**「必ずしも義務教育学校や施設一体型を目指す必要はない」**

## 4. 成功に共通する条件

### 準備期間
| 事例 | 準備期間 |
|------|---------|
| 信濃町 | 約7年 |
| 呉市 | 7年 |
| 三鷹市 | 約3年 |
| 早来（安平町） | 約4年 |

### 人口規模と成功事例の分布

| 人口規模 | 事例 | 特徴 |
|---------|------|------|
| ~10,000人 | 白川村、信濃町、安平町 | 選択肢なく統合は必然 |
| **10,000~50,000人** | **顕著な成功事例が少ない** | **須坂はここ** |
| 50,000~100,000人 | 綾瀬市 | 分離型で取組 |
| 100,000人~ | 三鷹市、呉市、品川区、つくば市 | モデルケースが多い |

## 5. 総合分析: 須坂の条件に当てはまるか

### 教育委員会の主張を裏付ける事実
1. 文科省調査で87%が「成果あり」と回答
2. 義務教育学校は全国261校に増加中
3. 施設一体型校では中1ギャップの緩和効果が多数報告
4. 信濃小中学校で学力向上が確認
5. 9年間カリキュラムは義務教育学校の方が制度的に作りやすい

### 留意点
1. 87%は自己評価であり、対照群との比較研究ではない
2. 施設分離型は一貫教育が形骸化するリスク
3. 成功事例の大半は小規模自治体か大都市。**須坂のような中規模都市の事例が乏しい**
4. 統合により教職員定数が減少する構造的問題
5. **三鷹市は「義務教育学校にせず、施設も統合せず」に成果を出している**
6. 信濃小中学校は人口8,000人・1中学校区で、須坂とは条件が大きく異なる
7. **人口5万人規模で4学園を同時に義務教育学校化した成功事例は全国に存在しない**
