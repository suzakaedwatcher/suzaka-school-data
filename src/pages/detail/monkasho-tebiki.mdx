# 文科省「手引き」改訂に関する調査報告

調査日: 2026-03-16

## 1. 改訂の経緯

### 調査研究協力者会議

「令和の日本型学校教育」を推進する学校の適正規模・適正配置の在り方に関する調査研究協力者会議。座長: 貞広斎子・千葉大学副学長。

| 回 | 日程 | 主な議題 |
|---|------|---------|
| 第1回 | 2025年3月5日 | 公立小中学校の適正規模・適正配置について |
| 第2回 | 2025年3月26日 | 委員発表 |
| 第3回 | 2025年4月21日 | 東京都町田市、ファインコラボレート研究所の発表 |
| 第4回 | 2025年6月2日 | これまでの議論の整理 |
| 第5回 | 2025年8月8日 | 通学環境、公共施設との複合化・共用化 |
| 第6回 | 2025年9月29日 | 小規模校として存続する場合の工夫 |
| 第7回 | 2025年12月1日 | 報告書最終まとめ骨子（案）提示 |
| **第8回** | **2026年1月16日** | **議論のまとめ素案を公表** |
| **第9回** | **2026年3月11日** | **議論のまとめ（案）を提示**（パブコメ結果反映） |

## 2. 改訂の3つの観点

| 観点 | 内容 |
|------|------|
| **広域化** | 周辺の市町村を巻き込んだ圏域で検討する |
| **総合化** | 首長部局も含めた各市町村全体でその地域の未来を考える視点で検討する |
| **現代化** | GIGAスクール構想等の変化に対応した学校教育の記載に改める |

## 3. 核心的な記述

### 結論ありきの排除

> 「学校の適正規模・適正配置の検討に当たっては、学校を統合する場合と小規模校として存続させる場合の一方を**予め結論ありきで評価することを避けなければならない**」

### 機械的適用の戒め

> 「通学距離や通学時間についての目安を示しつつも、各地域が抱える課題や実情は様々であることから、**機械的に手引の考え方を適用することは適当ではない**」

### 教育委員会の限界

> 「**教育委員会の所掌の範囲内での取組みで学校の適正規模・適正配置の検討を行うことが一層困難になることが見込まれる**」

### 区域外就学の活用

> 「**区域外就学・二地域居住の活用も小規模校で学ぶ児童生徒の学びの充実のための工夫として有効**」

松本市・山形県高畠町の事例を具体的に参照。

### こどもの意見反映

こども基本法に基づき「こどもや若者等の意見を反映させるために必要な措置を講ずること」を義務として記載。

## 4. 新たに充実が指示された事項

1. 地方公共団体の区域を超えた適正規模・適正配置の検討
2. 地域住民、他地域からの就学者、異年齢児童生徒との交流
3. **学年分校や統括校長の導入**（統合前の学校を分校として残す）
4. 教職員の育成（他の小規模校の教職員とのネットワーク構築）
5. 校務対応の効率化（共同学校事務室の設置）
6. GIGAスクール構想の活用
7. **統合の成果・課題の継続的な検証**（PDCAサイクルの導入）

## 5. 須坂学園構想との矛盾点

| 改訂の方向性 | 須坂の計画 | 矛盾の内容 |
|-----------|---------|---------|
| 結論ありきの評価を避ける | 統合を前提として4学園構想を策定 | 小規模校存続の選択肢が実質排除 |
| 機械的に基準を適用しない | 12-18学級に満たないことを再編の主要根拠に | 弾力規定を無視 |
| 広域化（圏域での検討） | 須坂市単独で4学園を完結 | 近隣自治体との連携を検討していない |
| 総合化（首長部局との連携） | 教育委員会主導の計画 | まちづくりの視点が不十分 |
| 区域外就学・二地域居住の活用 | 記載なし | 有効とされた手法が全く検討されていない |
| 学年分校の導入 | 仁礼小を廃校前提 | 分校として維持する選択肢が未検討 |
| 合意形成の丁寧なプロセス | 財政試算・スクールバス計画が「今後検討」 | 具体データなしで計画推進 |
| こどもの意見の反映 | 児童73%が「今のまま」だが反映されず | こども基本法の義務に反する |
| 統合後の検証 | 検証計画なし | PDCAサイクルの仕組みがない |
| 地域コミュニティの核 | 仁礼・豊丘から学校消滅 | コミュニティ維持策が未検討 |

## 6. 公表状況（2026年3月16日時点）

- **改訂手引きそのものはまだ未公表**
