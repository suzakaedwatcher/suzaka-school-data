# 5つの選択肢 — 30/40/50年で比較する

作成日: 2026-03-23

## 要約

学園構想（案A）は30年で73.5億円と公表されていますが、**使い続ける5校舎の長寿命化改修費68.5億円が含まれていません。** 実質コストは92〜119億円です。

5つの選択肢を40年・50年スパンで比較すると、**どの案でも2050年以降に再度の判断が必要**になります。「今決めれば終わり」という解は存在しません。

---

## 前提

- 児童生徒数: 3,500人(2026) → 2,300人(2040) → 1,800人(2050) → 1,200人(2065)
- 現在15校（小11＋中4）、全校が築31〜57年
- RC造の物理的寿命: 60〜80年
- 延床面積合計: 97,645 m²
- 財政調整基金: 約2億円
- 単価: 新築 35.41万円/m²、長寿命化改修 約21万円/m²（新築の6割）

---

## 案A: 現計画どおり — 分離型で15校→4学園

**やること:** 既存校舎をそのまま使い、組織だけ4学園に再編。段階的に廃校。

| 項目 | 内容 |
|------|------|
| 校舎 | 今のまま（築35〜57年の既存校舎を流用） |
| 新しい校舎 | なし |
| 総事業費 | 16億円（増改築のみ） |
| 廃校処理費 | 17.8億円（解体8校） |
| スクールバス | 18.3億円/30年 |
| 管理費削減 | ▲0.85億/年 |
| **30年総負担（公表値）** | **73.5億円（年2.5億）** |

**40年後:** 4校体制で1校300人。今の「小規模校」と同じ問題が再発。きれいな校舎はできない。

---

## 案Aの隠れたコスト: 残す校舎の改修費68.5億円

**30年総負担73.5億円には、使い続ける校舎の長寿命化改修費が含まれていません。**

### 残す校舎（4学園の核）

| 学園 | 校舎 | 築年数(2026) | 改修必要時期 | 改修費 |
|---|---|:---:|:---:|---:|
| 第一 | 高甫小 | 37年 | **2029年** | 13.7億 |
| 第一 | 東中 | 31年 | 2035年 | 13.7億 |
| 第二 | 常盤中 | 38年 | **2028年** | 13.7億 |
| 第三 | 相森中 | 36年 | **2030年** | 13.7億 |
| 第四 | 墨坂中 | 35年 | 2031年 | 13.7億 |
| | | | **合計** | **68.5億** |

5校とも**10年以内に長寿命化改修が必要**です。改修しなければ80年まで使えません。

### 案Aの真のコスト

| ケース | 計算 | 合計 |
|---|---|---:|
| 公表ベース | 73.5億（改修費なし） | 73.5億 |
| 統合関連として1/2補助が使える場合 | 73.5 + 18.8億 | **92.3億** |
| 通常の改修として1/3補助の場合 | 73.5 + 45.7億 | **119.2億** |

---

## 案B: 一体型新築 — 4校を新しく建てる

| 項目 | 内容 |
|------|------|
| 校舎 | 4校新築（1校あたり約6,000〜8,000m²） |
| 総事業費 | 120〜200億円 |
| 市の実質負担 | 33〜55億円 |
| 廃校処理費 | 22〜33億円（解体11校） |
| **30年総負担** | **100〜160億円** |

**40年後:** 新築校舎が「大規模な小規模校」に（1校300人に4校分の校舎）。財政調整基金2億円の須坂には初期投資が払えない。

---

## 案C: 長寿命化計画どおり — 15校を改修して使い続ける

| 項目 | 内容 |
|------|------|
| 校舎 | 既存15校を順次長寿命化改修 |
| 総事業費 | 205億円（15校×13.7億） |
| 市の実質負担 | 136.5億円（補助率1/3） |
| 市費教員追加 | 9億円/30年 |
| **30年総負担** | **204億円（年6.8億）** |

**40年後:** 15校は2040年代に明らかに過剰。ただし「いつ・どの校を減らすか」を状況に応じて判断できる柔軟性がある。

---

## 案D: 段階的再編 — 必要な統合だけ先にやる

| 項目 | 内容 |
|------|------|
| 統合対象 | 複式学級が見込まれる学校だけ（豊丘小→須坂小、仁礼小→日野小 など） |
| 統合費用 | 5〜10億円（受入校の増改築） |
| 残り12校の改修 | 109億円 |
| **30年総負担** | **130〜150億円（年4.3〜5.0億）** |

**40年後:** 人口動態に合わせて2,300人÷12校＝192人/校。まだ機能する。次に100人を切る学校が出たら、その都度判断。

---

## 案E: 仕組みで対応 — 統合せず教育の質を確保

**やること:** 校舎はそのまま。ICT・合同授業・教員兼務などソフト面で小規模校の課題に対応。

| 項目 | 内容 |
|------|------|
| 方法 | ICT遠隔授業・合同授業（山県方式）・教員兼務・小規模特認校 |
| 年間追加コスト | 0.5〜1.0億円/年 |
| 校舎 | 最低限の補修のみ（長寿命化改修はしない） |
| **30年総負担** | **80〜120億円** |

### 案Eのコスト内訳 — 代替案もタダではない

| 施策 | 年間コスト | 30年コスト | 備考 |
|---|---:|---:|---|
| 兼務発令 | 0円 | 0円 | 県教委の内部手続きのみ |
| 市費教員追加（5人） | 2,500万/年 | 7.5億 | 現在2人配置済。拡充する |
| ICT遠隔授業 | 初期数百万＋維持費 | 1〜3億 | GIGA端末は配備済 |
| 合同授業のバス運行 | 150万〜1.2億/年 | **4.5〜36億** | 距離・頻度で大幅に変動 |
| 小規模特認校の運営 | 200〜500万/年 | 1〜3億 | 広報費・通学支援費 |
| **小計: ソフト施策** | **0.5〜1.7億/年** | **14〜50億** | |
| 校舎の最低限補修 | 1.2〜2.3億/年 | 36〜69億 | 長寿命化改修はしない |
| 管理費（15校維持） | 1.95億/年 | 58.5億 | 現状の管理費をそのまま継続 |
| 管理費節約 | — | — | 統合しないので削減なし |

**合同授業のバス代が最大の変動要因です。** 週に何回・何校間で走らせるかで年間150万〜1.2億円と桁が変わります。山県方式（人口2.5万人・12校）では特定教科の合同授業のみバスを出すため、コストは限定的です。

**「安い」のではなく「性質が違う」:** 案Eは巨額の初期投資が不要で、年度ごとに調整可能です。効果が出なければやめられます。一方、統合は初期に数十億を投じて11校を廃校にする不可逆な判断です。

**40年後:** 合同頻度が高すぎる学校は自然に統合の議論に。校舎がまだあるので選択肢が残る。

## 30年/40年/50年 比較総括表

### 年平均負担（億円/年）

| | 30年平均 | 40年平均 | 50年平均 |
|---|---:|---:|---:|
| **A: 現計画（公表値）** | 2.5 | — | — |
| **A: 現計画（実質）** | **3.1〜4.0** | **2.6〜3.4** | **2.4〜3.3** |
| **B: 一体型新築** | 3.3〜5.3 | 2.8〜4.3 | 2.3〜3.5 |
| **C: 15校改修** | 6.8 | 5.5 | 4.6〜5.2 |
| **D: 段階的再編** | 4.3〜5.0 | 3.8〜4.4 | 3.3〜4.0 |
| **E: 仕組みで対応** | 2.7〜4.0 | 2.8〜3.9 | 2.8〜3.8 |

### この表から読み取れること

1. **案Aは「公表ベース」では最安だが、改修費を入れると案Eと大差ない。** 年平均3.1〜4.0億 vs 2.7〜4.0億。レンジが重なっている。

2. **50年スパンで見ると、全案が年2.3〜5.2億に収束する。** 「統合すれば安く済む」は30年の短期で見た場合の話。

3. **案Cが高いのは「15校全部を改修する」想定だから。** 実際には案Cと案Dの間に無数の中間解がある。

4. **最大のリスクは「案Aが安く見えたから選んだのに、10年後に改修費が来る」パターン。** 今の計画にはこの改修費が含まれていないため、市民は知らない。

5. **どの案でも2050年以降に再度の判断が必要。** 少子化が続く限り、「今決めれば終わり」という解は存在しない。

## 教育委員会に問うべきこと

**この5つの選択肢のうち、どれを選び、なぜそれが最善なのかを、比較データとともに示してください。**

現計画（案A）を選ぶなら:

- 使い続ける5校舎の改修費（68.5億）はいつ・いくらかかるのか
- なぜ計画に含まれていないのか
- 2068〜2075年に校舎が寿命を迎えた時どうするのか
- 40年後に4校で1校300人になった時どうするのか
