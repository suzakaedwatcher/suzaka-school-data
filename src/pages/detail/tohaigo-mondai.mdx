# 全国の学校統廃合 問題事例・反対事例集

調査日: 2026-03-16

## 1. 住民の反対で撤回・延期された事例

### 計画撤回・白紙化

**静岡県伊豆市**
- 3中学校統合の新設校計画を策定したが、住民の反発で白紙撤回（2017年）

**大阪府池田市**
- 小中一貫校4校計画 → 教員組合・保護者・住民が学区レベルで学習会を組織 → 一貫校は1校のみの開校にとどまり当初計画をほぼ阻止

**愛知県愛西市**
- 旧八開村地域の4小学校2中学校を統合する計画
- 旧八開村地域から小学校が全てなくなる計画
- 住民の約80%（約3,000人分）の反対署名
- 議論開始から10年経っても見通し立たず

**岩手県九戸村**
- 5小学校を統合し小中一貫校を建設する計画
- 2020年村長選で「見直し」公約の候補が当選、計画凍結

### 住民投票が行われた事例

**熊本県和水町** — 2度の住民投票（2013年、2016年）
**沖縄県宮古島市** — 住民投票条例案が市議会で否決（2018年）
**大阪府交野市** — 住民投票条例制定運動

### 現在進行中の反対運動

- **東京都東村山市** — 22校を14カ所に集約する計画に反対（2024年〜）
- **東京都三鷹市** — 小学校統合・複合化計画に反対（2023年〜）
- **広島県福山市** — 住民アンケートや署名を無視して計画推進

## 2. 統廃合後に問題が起きた事例

### 児童生徒への悪影響

**品川区（東京都）** — 半年間に3件の児童生徒自殺（2012年）
- 2月: 伊藤学園の小6女子
- 7月: 日野学園の中1男子
- 9月: 伊藤学園の中1男子
- 9年間同じ学校で人間関係のリセットが不可能なことが構造的要因

**つくば市（茨城県）** — 「小6問題」
- 施設一体型で中1ギャップは解消されたが、小6段階で新たな問題が発生
- 不登校児童生徒数が急増、2022年に茨城県の不登校率が全国トップに
- 教育長:「スクールバスに乗せて地域から切り離したら、社会力を育てる教育はできない」
- 「5校目の義務教育学校はつくらない」と方針転換

### 地域への悪影響

**京都府南山城村** — 統廃合後に高尾地区の人口が最も減少
**三重県御浜町** — 2003年統合開校、2015年度の児童生徒数は27名の極小規模校に
**京都市東山区** — 69億円投じても地域衰退は防げず

**三菱UFJリサーチ&コンサルティングの研究:**
- 高校が統廃合で消滅した市町村群では15-17歳人口層の格差が拡大
- 学校廃止 → 子育て世代の人口流出 → さらなる少子化の悪循環

### 通学問題

- スクールバス: 統合前156件 → 統合後325件（約2倍）
- つくば市・秀峰筑波: 全校約1,100名のうち約700名（64%）が20台のスクールバスで通学

### 財政的に想定外の負担

**山形県庄内町の試算:**
- 小学校5校・中学校2校維持: 約5,036万円
- 小学校1校・中学校1校に統合: 約6,923万円
- **統合した方がスクールバス関連費用でコスト増**

## 3. 小中一貫教育・義務教育学校の問題

### 実証研究の不在

- 一貫校と非一貫校を同一条件で比較した実証的な研究は存在しない
- 効果を主張する際のデータは小学生不登校率急上昇以前の2012年のものが多い
- 国立教育政策研究所:「中1ギャップという語に明確な定義はない」

### 施設分離型特有の問題

- 乗り入れ授業が実施困難
- 一貫教育が形骸化（実質6-3制のまま）
- 教員間の情報共有が不十分
