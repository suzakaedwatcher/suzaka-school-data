# 義務教育学校の全数調査

調査日: 2026-03-16

## 1. 義務教育学校の総数と年度別推移

### 最新データ（令和7年度/2025年5月1日現在）

文科省「令和7年度学校基本統計」確定値（2025年12月26日公表）より:

| 区分 | 学校数 |
|------|------|
| **義務教育学校** | **261校**（国立6、公立254、私立1） |
| 小学校 | 18,607校 |
| 中学校 | 9,827校 |
| 小中合計 | 28,434校 |

**全学校数に対する割合: 261 / (18,607 + 9,827 + 261) = 約0.91%**

### 勝山教育長の「232校で10倍に増えている」発言の検証

- 232校は令和6年度（2024年）の公立校数に近い数字（令和6年度の公立は232校）
- 制度開始の2016年度は22校（うち公立は21校）なので、22→232は確かに約10.5倍
- ただし最新の令和7年度は261校（公立254校）なので、公立だけで約12倍に相当

**しかし「10倍に増えた」という表現はミスリーディング:**
- 2016年に22校からスタートした「制度開始直後のゼロに近い数」からの倍率
- 絶対数としては全小中学校約28,700校のうち**わずか0.9%**
- 小学校は毎年200校以上減っている。義務教育学校の増加の相当部分は統廃合の結果

### 年度別推移（学校基本調査に基づく）

| 年度 | 学校数 | 前年度比 | 内訳（国立/公立/私立） |
|------|--------|---------|-----------------|
| 2016年（平成28年） | 22校 | -- | 4/17/1 |
| 2017年（平成29年） | 48校 | +26 | 4/43/1 |
| 2018年（平成30年） | 82校 | +34 | 4/77/1 |
| 2019年（令和元年） | 94校 | +12 | 5/88/1 |
| 2020年（令和2年） | 126校 | +32 | 5/120/1 |
| 2021年（令和3年） | 151校 | +25 | 5/145/1 |
| 2022年（令和4年） | 178校 | +27 | 5/172/1 |
| 2023年（令和5年） | 207校 | +29 | 5/201/1 |
| 2024年（令和6年） | 238校 | +31 | 5/232/1 |
| 2025年（令和7年） | 261校 | +23 | 6/254/1 |

**注目すべき点:**
- 私立は一貫して1校のみ（成果が認められるなら私立が追随するはずだが、していない）
- 増加の大半は公立であり、少子化に伴う統廃合の「受け皿」として使われている実態

## 2. 都道府県別設置状況

- **北海道: 33校**（最多）
- **鹿児島: 17校**
- 茨城、広島、新潟、福岡なども比較的多い
- **未設置県: 山梨、徳島、香川、愛媛、沖縄**（令和6年度時点）

北海道と鹿児島が多い理由: いずれも過疎地域が多く、小規模校の統廃合に伴い設置されるケースが多い。都市部での教育的理由による設置は少数。

## 3. 施設一体型 vs 施設分離型

国立教育政策研究所報告書より:

| 施設形態 | 割合 |
|---------|------|
| **施設一体型** | **約80%** |
| **施設隣接型** | **約4%** |
| **施設分離型** | **約4%** |
| その他・不明 | 残り |

**施設分離型は全国261校中わずか約10校程度**

### 施設分離型の具体例

| 学校名 | 自治体 | 分離の実態 |
|---------|--------|---------|
| 三戸学園 | 青森県三戸町 | 一体型校舎＋分離校舎（斗川小、5km以上離れた場所） |
| 西金沢学園 | 横浜市 | 本校舎（旧小学校）＋分校舎（旧中学校） |
| 品川区の一部校 | 東京都品川区 | 物理的に離れている |

### 須坂が構想する「1〜4年/5〜9年」分離型の前例

- **この形式の成功事例は確認できなかった**
- 三戸学園の分離校舎は実質的に6-3制のまま
- 品川区の分離型も「乗り入れ授業は実施困難」と報告

## 4. 成果検証データ

### 文科省の調査（平成27年）

- 実施自治体の23%が「大きな成果」、76%が「成果あり」と回答

**この調査の問題点:**
- **自己評価**であり、対照群との比較がない
- 設置者自身が「成果なし」と答えるインセンティブがない
- 客観的な学力テストデータによる検証ではない

### 国立教育政策研究所の見解

- **「中1ギャップという語に明確な定義はなく、その前提となっている事実認識も客観的事実とは言い切れない」**
- 小6から中1への不登校増加率は1.3倍前後であり、「必ずしも急増とは言えない」

### 施設分離型特有の課題

| 項目 | 施設一体型 | 施設分離型 |
|------|----------|----------|
| 乗り入れ授業 | 日常的に実施可能 | **困難** |
| 合同研修 | 容易 | **頻度低下** |
| 異学年交流 | 日常的 | **イベント時のみ** |
| 一貫教育の実感 | あり | **希薄** |
| 教員間の情報共有 | 容易 | **片方の校舎の情報が入らない** |
